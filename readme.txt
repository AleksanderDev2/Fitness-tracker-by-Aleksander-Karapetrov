ОПИСАНИЕ:
  Конзолно приложение на Python за проследяване на тренировки
  и изгорени калории. Позволява добавяне на упражнения,
  сортиране по калории, филтриране по мускулна група и
  обобщена статистика.

СТРУКТУРА НА ПРОЕКТА:
  Fitness-tracker-by-Aleksander-Karapetrov/
  ├── models.py          — клас FitnessTracker с методи
  ├── main.py            — главна програма (демонстрация)
  ├── requirements.txt   — използвани библиотеки
  └── readme.txt         — документация (този файл)

ИЗИСКВАНИЯ:
  - Python 3.8 или по-нова версия
  - Не са нужни външни библиотеки

СТАРТИРАНЕ:
  1. Клонирайте хранилището:
       git clone https://github.com/<user>/Fitness-tracker-by-Aleksander-Karapetrov.git

  2. Влезте в папката:
       cd Fitness-tracker-by-Aleksander-Karapetrov

  3. Стартирайте програмата:
       python main.py

КЛАС FitnessTracker (models.py):
  __init__(потребител)          — създава обект с потребителско име
  добави_тренировка(...)        — добавя упражнение с параметри
  общо_калории()                — връща сумата на всички калории
  сортирай_по_калории()         — връща сортиран списък (низходящо)
  филтрирай_по_група(група)     — филтрира по мускулна група
  обобщение_по_групи()          — речник с калории по група
  __str__()                     — текстово представяне на обекта

ИЗПОЛЗВАНИ БИБЛИОТЕКИ:
  - datetime (вградена) — за запис на датата на всяка тренировка
